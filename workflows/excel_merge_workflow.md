# Excel File Merge Workflow

## Goal
Automatically merge multiple Excel files into a single consolidated file.

## Input
- Multiple Excel files with similar structure
- Files located in a single folder

## Automation Steps
1. Detect all Excel files in a directory
2. Read each file into memory
3. Combine all data into one dataset
4. Export merged output to Excel

## Output
- One consolidated Excel file

## Use Case
Useful for monthly reports, multi-source data collection,
and combining team submissions.

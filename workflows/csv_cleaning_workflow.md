# CSV Cleaning Automation Workflow

## Goal
Convert messy CSV data into a clean, structured Excel file automatically.

## Input
- One or more raw CSV files
- Unstructured or inconsistent column names
- Missing or duplicate values

## Automation Steps
1. Load CSV file using Python
2. Normalize column names
3. Remove duplicate rows
4. Handle missing values
5. Export cleaned data to Excel format

## Output
- Clean Excel file ready for analysis or reporting

## Use Case
Ideal for businesses and individuals dealing with messy CSV exports
from websites, tools, or manual data entry.

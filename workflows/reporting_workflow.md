# Automated Reporting Workflow

## Goal
Generate a simple summary report from cleaned Excel data.

## Input
- Cleaned Excel file
- Structured rows and columns

## Automation Steps
1. Load cleaned Excel data
2. Calculate basic metrics
3. Generate summary report
4. Export report to Excel

## Output
- Excel report with key metrics

## Use Case
Helpful for quick insights, management reports,
and automated data summaries.

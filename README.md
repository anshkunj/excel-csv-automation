# Excel & CSV Automation Workflows

This repository contains practical Excel and CSV automation workflows built using Python.
The goal is to eliminate repetitive manual spreadsheet work and create clean, automated,
and reusable data processes.

These examples reflect real-world tasks such as data cleaning, file merging,
formula automation, and report generation.

---

## What This Repository Demonstrates

- Excel and CSV automation using Python  
- Data cleaning and formatting  
- Automating repetitive spreadsheet tasks  
- Generating Excel reports from raw CSV data  
- Business-focused, reusable automation scripts  

---

## Example Automations Included

1. Cleaning messy CSV files and exporting clean Excel sheets  
2. Merging multiple Excel or CSV files automatically  
3. Applying formulas and calculations programmatically  
4. Generating summary reports from raw data
